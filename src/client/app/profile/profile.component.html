Oblong is a text mining service that collates expertise information from Academics at Universities from all the different databases of Academic Publications that exist on the web.

This service is designed to be easy to intuitive and easy to use, example of search queries include:

  • Find me a researcher with expertise in Robotics
  • I am looking for a member of staff who has recently written about OsteoArthritus

Features

  • Ready to go, statically typed build system using Gulp for working with TypeScript.
  • Production and development builds.
  • Sample unit tests with Jasmine and Karma including code coverage via Istanbul.
  • End-to-end tests with Protractor.
  • Development server with live reload.
  • TypeScript definition management using Types.
  • Basic Service Worker, which implements "Cache then network strategy".
